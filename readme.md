# Setup

    npm install

# Developing locally

    npm run dev

Go to http://localhost:8081/src/static.  
Activate the [LiveReload][livereload] extension.

[livereload]: https://chrome.google.com/webstore/detail/livereload/jnihajbhpnppcggbcgedagnkighmdlei?hl=en

# Production

    npm run build
    npm start
